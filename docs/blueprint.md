# **App Name**: Credit Clarity

## Core Features:

- Card Registration: Allow users to register multiple credit cards to manage.
- Installment Tracking: Track and display the status of each purchase with installments, indicating when they will end.
- Debt Tracking: Record debts to third parties to manage accounts payable for paying the card bill.
- Purchase Categorization: Automatically categorize purchases (e.g., food, transportation) and provide a summary.
- Spending insights: Display charts to show expenses evolution using an external charting library.
- Google Sheets Integration: Synchronize card data with Google Sheets to use it as storage.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for trustworthiness and stability, reflecting financial management.
- Background color: Very light blue (#E8EAF6), near-white, providing a clean and calm backdrop.
- Accent color: A purple (#7E57C2) is used to call attention.
- Body and headline font: 'Inter' (sans-serif) for a clean, modern, and easily readable interface.
- Use flat, consistent icons related to finance and credit cards.
- A responsive design that adapts seamlessly to both desktop and mobile devices, ensuring accessibility across platforms.
- Subtle animations for loading data and transitions to maintain user engagement without distraction.
